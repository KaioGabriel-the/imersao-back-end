# 😎 Imersão back-end Alura
Este é um projeto simples que foi desenvolvido durante a imersão back-end da Alura.

## 🚀 Funcionalidades

- **Servidor em Node.js**: Implementação de um servidor local utilizando o framework Express.
- **API RESTful**: Criação de endpoints para manipulação de dados relacionados às condições climáticas de diferentes cidades.
- **Integração com MongoDB**: Armazenamento e recuperação de dados em um banco de dados MongoDB.
- **Exibição de Resultados**: Apresentação das condições climáticas das cidades consultadas.

## 🛠 Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)

## ⚙️ Como Executar o Projeto

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/KaioGabriel-the/imersao-back-end.git
   cd imersao-back-end/back-end ```
   
2. **Instale as dependências**

   ```bash
    npm install 
3. **Configure as variáveis de ambiente**
     ```bash
    MONGODB_URI=seu_uri_do_mongodb
    PORT=3000
4. **Inicie o servido**
     ```bash
    npm start
## 📄 Endpoints da API

- `GET /posts`: Retorna todas as publicações da rede social.
- `POST /posts`: Cria uma nova publicação.
- `GET /posts/:id`: Retorna uma publicação específica pelo ID.
- `PUT /posts/:id`: Atualiza uma publicação existente.
- `DELETE /posts/:id`: Remove uma publicação específica.
